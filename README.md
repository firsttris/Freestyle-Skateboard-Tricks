## Session Goals

- Land all basics in **FS + BS** consistently
- 3 clean endovers in a row
- 10 seconds of smooth spacewalk without stepping off
- Film one short line: `Tic-Tac -> One-Foot Tic-Tac -> Endover -> Smoothie`
